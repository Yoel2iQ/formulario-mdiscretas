7. ÁLGEBRA BOOLEANA

7.1 ¿Qué es el álgebra booleana?

Es un método para simplificar circuitos lógicos por medio de dos estados: True y False, es decir, verdadero y falso, donde el 1 equivale a verdadero y 0 equivale a falso.

7.2 Leyes del álgebra Booleana

Leyes conmutativas

𝐴 + 𝐵 = 𝐵 + 𝐴

𝐴 ∙ 𝐵 = 𝐵 ∙ 𝐴

Leyes asociativas

(𝐴 + 𝐵) + 𝐶 = 𝐴 + (𝐵 + 𝐶)

(𝐴 ∙ 𝐵) ∙ 𝐶 = 𝐴 ∙ (𝐵 ∙ 𝐶)

Leyes distributivas

𝐴 ∙ (𝐵 + 𝐶) = (𝐴 ∙ 𝐵) + (𝐴 ∙ 𝐶)

𝐴 + (𝐵 ∙ 𝐶) = (𝐴 + 𝐵) ∙ (𝐴 + 𝐶)

7.3 Otras identidades útiles

A + (A ∙ B) = A

A ∙ (A + B) = A

A + (A ∙ B) = A + B

(A + B) ∙ (A + B) = A

(A + B) ∙ (A + C) = A + (B ∙ C)

A + B + (A ∙ B) = A + B

(A ∙ B) + (B ∙ C) + (B ∙ C) = (A ∙ B) + C

(A ∙ B) + (A ∙ C) + (B ∙ C) = (A ∙ B) + (B ∙ C)
